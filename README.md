En tanto backend como fronend con NodeJs instalado lo unico que hay que hacer es
npm install y npm run en cada uno de estos para que empiezen a funcionar, primero iniciando el backend
y luego el frontend.

esta pagina es una pagina donde los usuarios pueden subir sus propios juegos para poder jugarlo en la misma sin tener que recurrir a descargas, en cada uno de los juegos estan commentarios con su calificacion de calidad y dificultad para que las demas personas lo sepan, esto reforzado con el tiempo de juego de cada usuario en el juego referido, y si es que tus comentarios son muy diferentes en general de el promedio del juego la IA de gemini flash apoyara tu mensaje agradeciendo diferentes puntos de vista, cada usuario puede modificar todos sus juegos a placer y eliminarlos cuando estos quieran.
